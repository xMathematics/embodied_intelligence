## 一、问题背景

### 1.1 为什么需要Bundle Adjustment？

在BA之前，三维重建方法存在以下问题：

| 问题 | 传统方法 | 局限性 |
|------|---------|-------|
| **精度** | 分步估计 | 误差累积 |
| **一致性** | 独立优化 | 缺乏全局一致性 |
| **效率** | 直接求解 | 计算复杂度高 |

### 1.2 研究现状

- **分步方法**：先估计位姿，再重建结构
- **独立优化**：分别优化相机和点
- **缺乏统一框架**：没有全局优化

### 1.3 论文要解决的核心问题

1. **如何统一优化相机位姿和三维结构？**
2. **如何高效求解大规模BA问题？**
3. **如何处理BA的数值稳定性问题？**

## 二、整体架构

### 2.1 BA问题定义

**目标函数**：

$$\min_{\mathbf{x}} \sum_{i=1}^{m} \sum_{j \in \mathcal{J}_i} \| \mathbf{e}_{ij}(\mathbf{x}) \|^2$$

其中：
- $\mathbf{x}$：状态向量（相机位姿 + 三维点）
- $\mathbf{e}_{ij}$：第i个相机观察第j个点的重投影误差
- $\mathcal{J}_i$：第i个相机观察到的点集合

### 2.2 问题规模

| 参数 | 数量 | 说明 |
|------|------|------|
| 相机数量 | n | 每个相机6个参数（3位置+3姿态） |
| 点数量 | m | 每个点3个参数（x,y,z） |
| 观测数量 | k | 每个观测2个参数（u,v） |

**典型规模**：
- 相机：几十到几百
- 点：几千到几万
- 观测：几万到几十万

### 2.3 优化流程

```
┌─────────────────────────────────────────────────────┐
│                  Bundle Adjustment                 │
├─────────────────────────────────────────────────────┤
│  输入：初始位姿和点估计                              │
│  ├─ 相机位姿 (R, t)                                │
│  └─ 三维点坐标 (X, Y, Z)                           │
├─────────────────────────────────────────────────────┤
│  迭代优化                                          │
│  ├─ 1. 计算重投影误差                              │
│  ├─ 2. 构建雅可比矩阵                              │
│  ├─ 3. 求解线性方程组                              │
│  └─ 4. 更新状态向量                                │
├─────────────────────────────────────────────────────┤
│  输出：优化后的位姿和点                              │
│  ├─ 优化后的相机位姿                               │
│  └─ 优化后的三维点坐标                             │
└─────────────────────────────────────────────────────┘
```

## 三、设计思路

### 3.1 重投影误差

**投影模型**：

$$\mathbf{u} = \pi(\mathbf{P}, \mathbf{X})$$

其中：
- $\mathbf{u} = (u, v)$：图像坐标
- $\mathbf{P}$：相机投影矩阵
- $\mathbf{X} = (X, Y, Z, 1)$：齐次坐标
- $\pi$：投影函数

**重投影误差**：

$$\mathbf{e} = \mathbf{u}_{\text{obs}} - \pi(\mathbf{P}, \mathbf{X})$$

### 3.2 雅可比矩阵

**链式法则**：

$$\frac{\partial \mathbf{e}}{\partial \mathbf{x}} = \frac{\partial \mathbf{e}}{\partial \mathbf{u}} \cdot \frac{\partial \mathbf{u}}{\partial \mathbf{x}}$$

**块结构**：
- 相机参数块
- 点参数块
- 稀疏结构（每个点只被少数相机观察）

### 3.3 线性求解

**正规方程**：

$$\mathbf{H} \Delta \mathbf{x} = \mathbf{b}$$

其中：
- $\mathbf{H} = \mathbf{J}^T \mathbf{J}$：海森矩阵
- $\mathbf{b} = -\mathbf{J}^T \mathbf{e}$：梯度

**稀疏性利用**：
- 海森矩阵是稀疏的
- 使用稀疏矩阵求解器
- 分块对角近似

### 3.4 收敛策略

**阻尼最小二乘**：

$$(\mathbf{H} + \lambda \mathbf{I}) \Delta \mathbf{x} = \mathbf{b}$$

**信任区域方法**：
- 维护信任区域半径
- 根据残差减少量调整半径
- 保证收敛稳定性

## 四、核心创新点

### 4.1 统一优化框架

**为什么重要**：
- 将相机位姿和三维结构统一优化
- 避免分步估计的误差累积
- 提高整体精度

**优势**：
- 全局最优解
- 一致性保证
- 精度提升

### 4.2 稀疏性利用

**为什么重要**：
- BA问题规模大
- 直接求解计算量太大
- 需要利用稀疏结构

**实现方式**：
- 稀疏矩阵存储
- 分块求解
- 减少内存占用和计算量

### 4.3 数值稳定性

**为什么重要**：
- BA问题可能病态
- 数值不稳定会导致发散
- 需要稳定的求解方法

**处理策略**：
- 阻尼最小二乘
- 信任区域方法
- 正则化

## 五、解决的问题

### 5.1 精度提升

**对比分步方法**：

| 方法 | 精度 | 说明 |
|------|------|------|
| 分步估计 | 较低 | 误差累积 |
| Bundle Adjustment | 较高 | 全局优化 |

**实验结果**：
- 在相同数据下，BA可以将误差降低50%以上
- 是三维重建的标准方法

### 5.2 效率提升

**稀疏求解**：
- 时间复杂度从O(n^3)降低到O(n)
- 能够处理大规模问题
- 实际应用中效率提升100倍以上

### 5.3 稳定性提升

**数值稳定**：
- 能够处理病态问题
- 收敛性得到保证
- 鲁棒性增强

## 六、缺陷与局限性

### 6.1 计算复杂度

**问题**：
- 虽然利用了稀疏性，但大规模问题仍然耗时
- 需要高性能计算资源
- 难以实时处理

**原因**：
- 每次迭代需要构建和求解大型线性系统
- 数据量大时内存占用高

### 6.2 局部最优

**问题**：
- BA是局部优化方法
- 初始估计不好会陷入局部最优
- 需要良好的初始化

**原因**：
- 目标函数是非凸的
- 存在多个局部极小值

### 6.3 离群点敏感

**问题**：
- 对异常值敏感
- 单个错误观测会影响整体结果
- 需要预处理去除离群点

**原因**：
- L2范数对大误差敏感
- 需要鲁棒估计方法

## 七、实验结果

### 7.1 数据集

- 重建精度显著提升
- 能够处理大规模场景
- 收敛稳定
