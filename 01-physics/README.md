
# 物理基础模块

## 模块概述

物理知识是理解机器人运动和环境交互的基础，本模块涵盖经典力学、运动学、动力学等核心内容。

## 学习目标

完成本模块学习后，您将能够：
- 理解刚体运动学原理
- 掌握机器人动力学模型
- 理解控制理论基础

## 学习路径

### 第一部分：经典力学 ✅（1周）

| 章节 | 内容 | 难度 | 链接 |
|------|------|------|------|
| 1.1 | 牛顿力学 | ⭐⭐ | [查看](classical-mechanics/01-newtonian-mechanics.md) |
| 1.2 | 刚体运动 | ⭐⭐⭐ | [查看](classical-mechanics/02-rigid-body-motion.md) |

**核心内容：**
- 牛顿三大定律、力与加速度
- 动量与能量守恒
- 刚体的平动与转动
- 角动量、惯性张量
- Euler动力学方程

**在具身智能中的应用：**
- 机器人动力学建模
- 物理仿真
- 姿态控制
