# 数学基础模块

> 具身智能的数学基石 - 从零开始的系统学习

## 模块概述

数学是具身智能的基石，本模块从零开始系统讲解具身智能所需的全部数学知识，包括线性代数、微积分、概率论、数值计算、几何基础、优化理论和信息论等核心内容。

## 学习目标

完成本模块学习后，您将能够：
- 掌握线性代数的核心概念和运算
- 理解微积分和向量微积分
- 掌握概率论和统计推断方法
- 理解优化理论和数值方法
- 掌握几何基础和变换
- 理解信息论和编码

### 必读教材
1. **线性代数**
   - 《线性代数及其应用》- Gilbert Strang
   - 《Introduction to Linear Algebra》- Strang
   - 《Numerical Linear Algebra》- Trefethen & Bau

2. **微积分**
   - 《Calculus》- James Stewart
   - 《数学分析》- 陈纪修
   - 《Vector Calculus》- Marsden & Tromba

3. **优化理论**
   - 《Convex Optimization》- Boyd & Vandenberghe
   - 《Numerical Optimization》- Nocedal & Wright

4. **概率论**
   - 《Probability and Statistics》- DeGroot
   - 《Pattern Recognition and Machine Learning》- Bishop

5. **信息论**
   - 《Elements of Information Theory》- Cover & Thomas

### 在线课程
1. **MIT 18.06** - Linear Algebra (Gilbert Strang)
2. **MIT 18.02** - Multivariable Calculus
3. **Stanford EE364A** - Convex Optimization
4. **Khan Academy** - 概率与统计

### 编程工具
- **NumPy** - 矩阵运算和科学计算
- **SciPy** - 科学计算库
- **SymPy** - 符号计算
- **Matplotlib** - 可视化
- **PyTorch/TensorFlow** - 张量运算

## 学习评估

### 自测清单

#### 线性代数
- [ ] 能够手动计算矩阵乘法和逆矩阵
- [ ] 理解SVD分解的几何意义
- [ ] 能够计算特征值和特征向量
- [ ] 理解向量空间和线性变换
- [ ] 掌握张量的基本概念

#### 微积分
- [ ] 能够计算各种函数的导数
- [ ] 理解多元函数的偏导数和梯度
- [ ] 掌握梯度、散度、旋度的计算
- [ ] 理解优化理论的基本方法
- [ ] 能够实现梯度下降和牛顿法

#### 概率论
- [ ] 理解条件概率和贝叶斯定理
- [ ] 掌握常见概率分布
- [ ] 理解期望和方差
- [ ] 掌握参数估计方法

### 实践项目

#### 基础项目
1. **PCA降维** - 使用SVD实现主成分分析
2. **梯度下降** - 实现各种优化算法
3. **图像压缩** - 使用SVD进行图像压缩

#### 进阶项目
4. **卡尔曼滤波** - 实现状态估计算法
5. **ICP配准** - 实现点云配准算法
