# 通信接口

## 概述

本模块介绍机器人常用的通信协议和接口标准。

## 内容列表

### 6.1 串行通信
- UART（串口）
- I2C总线
- SPI总线
- RS-232/RS-485

### 6.2 工业总线
- CAN总线
- CAN FD
- EtherCAT
- Modbus

### 6.3 网络通信
- Ethernet
- TCP/IP协议
- UDP通信
- ROS通信

### 6.4 无线通信
- WiFi
- 蓝牙
- LoRa
- Zigbee

### 6.5 接口标准
- USB接口
- GPIO
- ADC/DAC
- HDMI/DisplayPort
