# 机器人硬件模块

## 模块概述

机器人硬件是具身智能的物理载体，本模块从机械结构开始，涵盖传感器、执行器、嵌入式系统、电源管理等核心硬件组件，为理解机器人系统提供全面的硬件知识框架。

## 学习目标

完成本模块学习后，您将能够：
- 理解机器人机械结构设计原理
- 熟悉各类传感器的工作原理和应用
- 掌握执行器和驱动系统
- 了解嵌入式系统和硬件接口
- 理解电源系统设计
- 熟悉常见机器人硬件平台
