
# 机器人硬件模块

## 模块概述

机器人硬件是具身智能的物理载体，本模块涵盖机器人结构、传感器、执行器等内容。

## 学习目标

完成本模块学习后，您将能够：
- 理解机器人硬件组成
- 了解传感器与执行器原理
- 掌握硬件接口与通信

## 学习路径

### 第一部分：机器人结构（1周）

#### 1.1 机械结构

**核心概念：**
- 串联与并联机构
- 关节类型（旋转、移动）
- 连杆设计

**推荐资源：**
- 《Robotics, Vision and Control》- Corke
- 《Mechanism Design》- Norton

**实践练习：**
- 理解不同机器人构型

#### 1.2 机器人分类

**核心概念：**
- 工业机械臂
- 移动机器人
- 人形机器人

**实践练习：**
- 研究不同类型机器人的应用场景

---

### 第二部分：传感器（1周）

#### 2.1 视觉传感器

**核心概念：**
- RGB相机
- RGB-D相机（Kinect、RealSense）
- 事件相机

**推荐资源：**
- 《Computer Vision: Algorithms and Applications》- Szeliski

**实践练习：**
- 使用RGB-D相机获取深度图像

#### 2.2 距离传感器

**核心概念：**
- 激光雷达（LiDAR）
- 超声波传感器
- TOF相机

**实践练习：**
- 理解LiDAR数据格式

#### 2.3 惯性测量单元

**核心概念：**
- 加速度计
- 陀螺仪
- 传感器融合

**实践练习：**
- 理解IMU数据处理

---

### 第三部分：执行器与驱动（1周）

#### 3.1 电机类型

**核心概念：**
- 直流电机
- 步进电机
- 伺服电机

**实践练习：**
- 理解不同电机的特点

#### 3.2 驱动系统

**核心概念：**
- 电机驱动电路
- 编码器
- 减速器

**实践练习：**
- 理解伺服控制原理

---

### 第四部分：硬件接口（1周）

#### 4.1 通信协议

**核心概念：**
- ROS通信
- CAN总线
- 串口通信

**推荐资源：**
- ROS官方文档

**实践练习：**
- 使用ROS进行节点通信

#### 4.2 嵌入式系统

**核心概念：**
- Arduino
- Raspberry Pi
- FPGA

**实践练习：**
- 编写简单的嵌入式控制程序
