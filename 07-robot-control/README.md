# 机器人控制模块

## 模块概述

机器人控制是具身智能的执行层，负责将感知和决策转化为机器人的实际动作。本模块从基础控制理论开始，涵盖运动控制、力控制、自适应控制、学习控制等多个方向，提供广度优先的知识覆盖。

## 学习目标

完成本模块学习后，您将能够：
- 理解基础控制理论和方法
- 掌握机器人运动控制技术
- 了解力控制和柔顺控制原理
- 熟悉自适应控制和鲁棒控制方法
- 了解学习控制和智能控制方法
- 掌握ROS控制框架的使用
