# 基础控制理论

## 概述

本模块介绍机器人控制的基础理论，包括PID控制、状态空间控制等核心方法。

## 内容列表

### 1.1 控制理论基础
- 控制系统分类（开环/闭环）
- 反馈控制原理
- 稳定性分析
- 性能指标

### 1.2 PID控制
- 比例-积分-微分控制
- 参数整定方法（Ziegler-Nichols）
- PID变种（PD、PI、I-PD）
- 抗积分饱和

### 1.3 状态空间控制
- 状态空间表示
- 可控性与可观性
- 状态反馈
- 状态观测器

### 1.4 LQR控制
- 线性二次调节器
- 代价函数设计
- Riccati方程求解
- 最优控制

### 1.5 极点配置
- 极点配置方法
- Ackermann公式
- 动态响应设计
