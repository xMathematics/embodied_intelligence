# 计算机视觉项目

## 项目概述

本项目包含计算机视觉基础实践练习，涵盖图像处理、特征提取和目标检测。

## 实践内容

### 1. 图像处理
- 图像读取与保存
- 图像滤波
- 边缘检测
- 形态学操作

### 2. 特征提取
- SIFT特征
- SURF特征
- ORB特征
- 直方图特征

### 3. 目标检测
- Haar级联检测器
- HOG特征+SVM
- 模板匹配

## 学习目标

完成本项目后，您将能够：
- 使用OpenCV进行图像处理
- 提取图像特征
- 实现基本的目标检测系统
