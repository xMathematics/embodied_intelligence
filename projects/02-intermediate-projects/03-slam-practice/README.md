# SLAM实践项目

## 项目概述

本项目包含SLAM基础实践练习，涵盖视觉里程计、后端优化和三维重建。

## 实践内容

### 1. 视觉里程计
- 特征匹配
- 相机位姿估计
- 三角测量
- 光束法平差

### 2. SLAM后端
- 扩展卡尔曼滤波
- 粒子滤波
- 因子图优化

### 3. 三维重建
- 稀疏重建
- 稠密重建
- 点云处理

## 学习目标

完成本项目后，您将能够：
- 理解SLAM基本原理
- 使用OpenCV实现视觉里程计
- 处理SLAM数据集
