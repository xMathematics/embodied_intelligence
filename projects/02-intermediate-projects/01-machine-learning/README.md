# 机器学习项目

## 项目概述

本项目包含经典机器学习算法的实践练习，涵盖监督学习、无监督学习和模型评估。

## 实践内容

### 1. 监督学习
- 线性回归
- 逻辑回归
- 支持向量机
- 决策树与随机森林
- K近邻算法

### 2. 无监督学习
- K-Means聚类
- 层次聚类
- 主成分分析
- 关联规则挖掘

### 3. 模型评估
- 交叉验证
- 评估指标计算
- 模型选择
- 超参数调优

## 学习目标

完成本项目后，您将能够：
- 实现经典机器学习算法
- 使用Scikit-learn进行模型训练
- 评估和优化机器学习模型
