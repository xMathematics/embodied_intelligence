# 深度学习项目

## 项目概述

本项目包含深度学习实践练习，涵盖CNN、Transformer、生成模型等核心架构。

## 实践内容

### 1. CNN基础
- 卷积神经网络实现
- 经典架构（LeNet, AlexNet, ResNet）
- 图像分类
- 语义分割

### 2. Transformer
- 自注意力机制
- Transformer架构
- BERT实现
- ViT（视觉Transformer）

### 3. 生成模型
- 生成对抗网络
- 变分自编码器
- Diffusion模型

## 学习目标

完成本项目后，您将能够：
- 使用PyTorch实现深度学习模型
- 训练和优化神经网络
- 理解前沿深度学习架构
