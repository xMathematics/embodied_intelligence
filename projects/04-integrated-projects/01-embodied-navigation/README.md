# 具身导航系统项目

## 项目概述

本项目是一个完整的具身导航系统，集成感知、规划和控制模块，在仿真环境中实现自主导航。

## 实践内容

### 1. 感知模块
- 视觉感知
- 深度估计
- 语义分割
- 环境理解

### 2. 规划模块
- 全局路径规划
- 局部避障
- 运动规划
- 行为决策

### 3. 控制模块
- 运动控制
- 轨迹跟踪
- 姿态控制
- 闭环控制

## 学习目标

完成本项目后，您将能够：
- 理解具身导航系统架构
- 在Habitat/AI2-THOR中实现导航
- 集成感知-规划-行动闭环
