# 机器人学模块

## 模块概述

机器人学是具身智能的核心载体，本模块涵盖机器人的基础概念、运动学、动力学、感知、规划等核心内容，旨在提供机器人学的全面知识框架。

## 学习目标

完成本模块学习后，您将能够：
- 理解机器人学的基本概念和分类
- 掌握机器人运动学原理（正运动学、逆运动学）
- 理解机器人动力学建模方法
- 了解机器人感知系统和传感器技术
- 熟悉路径规划和运动控制基础
- 了解不同类型机器人的应用场景
